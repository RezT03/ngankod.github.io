While Loop

while loop akan terus menjalankan blok kode selama kondisi yang ditentukan bernilai benar.
syntax :
while(kondisi){
//blok kode
}
Contoh :
int i=0
while(i<5){
cout << i;
i++;
}
kode diatas akan menghasilkan output : 01234
