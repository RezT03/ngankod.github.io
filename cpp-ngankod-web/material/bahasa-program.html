Bahasa Pemrograman

Bahasa pemrograman ini merupakan suatu himpunan dari aturan sintaks dan semantik yang dipakai untuk mendefinisikan program komputer. Bahasa ini memungkinkan seorang programmer dapat menentukan secara persis data mana yang akan diolah oleh komputer, bagaimana data ini akan disimpan/diteruskan, dan jenis langkah apa secara persis yang akan diambil dalam berbagai situasi.
Berdasarkan tingkatannya bahasa pemrograman dibagi menjadi 3 level yaitu :
  1. Bahasa tingkat rendah (Low-Level Language)
    2.  Bahasa program tingkat rendah berisi instruksi-instruksi yang ditujukan kepada komputer dengan menggunakan kode biner (binary). Kode-kode tersebut akan langsung diterjemahkan oleh komputer tanpa harus melalui proses kompilasi. Contohnya adalah Assembly
  2. Bahasa Tingkat menengah (Mid-Level Language)
    3.  Bahasa program tingkat menengah adalah instruksi yang berupa kode mnemonic seperti ADD, SUB, DIV, STOLOD, JMP dan lainnya. Namun, bahasa dalam tingkat menengah ini harus diterjemahkan terlebih dahulu ke dalam bahasa mesin, karena komputer hanya mengerti penggunaan bahasa mesin. Contohnya adalah C
  3. Bahasa tingkat tinggi (High-Level Language)
    4.  Bahasa program tingkat tinggi ini berisi instruksi-instruksi yang menggunakan bahasa alamiah yang dimengerti manusia, seperti bahasa Inggris atau matematika. Tingkatan ini diciptakan untuk mengatasi kekurangan yang dimiliki tingkat menengah dan tingkat rendah. Contohnya adalah C++.
Sedangkan Berdasarkan cara mengeksekusi kode program dibedakan menjadi 2, yaitu Compiled dan Interpreted.

Perbedaan Compiled dan Interpreted Language

Untuk menjalankan kode yang sudah dibuat, komputer menggunakan 2 cara yaitu Compiled dan Interpreted. Nah apa sih perbedaan keduanya?

Compiled Language

Compiled Language adalah bahasa program yang mana untuk menjalankannya diperlukan software bernama Compiler. Compiler adalah program untuk menerjemahkan source code ke dalam bahasa mesin. Apabila ada kesalahan penulisan kode program maka compiler akan mengirimkan kode error ketika mencoba menjalankan kode program. Namun jika tidak ada kesalahan dalam kode program, compiler akan menghasilkan sebuah executable file, contohnya jika menjalankan kode program C++ di Windows menghasilkan file .exe, dan jika dijalankan di MacOS akan menghasilkan file .o
Contoh bahasa : C++, Java, C.

Interpreted Language

Interpreted Language adalah bahasa pemrograman yang menjalankannya menggunakan Interpreter. Interpreter adalah program penerjemah source code secara langsung tanpa perlu menyusunnya menjadi executable file terlebih dahulu. untuk menjalankan program perlu membuka source code.
Contoh bahasa : Python, Javascript, php
Baik Compiled maupun Interpreted language, keduanya ditulis di sebuah text editor/code editor yang biasanya terdapat di sebuah IDE.

Apa itu IDE dan Text Editor?

IDE (Integrated Software Environtment) : software yang memiliki banyak fitur untuk membuat software. Didalam IDE terdapat Text editor, Syntax highlighting, Compiler, dan debugger. IDE biasanya digunakan untuk pemrograman yang bersifat Compiled bukan Interpreted. Namun saat ini Text editor bisa bekerja seperti IDE dengan bantuan extension yang dikembangkan oleh developer text editor itu sendiri ataupun oleh pihak ketiga.
Text editor : software untuk menampilkan text murni tanpa adanya formatiing apapun seperti bold,italic, dsb, serta digunakan untuk mengedit text tersebut.
Contoh IDE : Dev C++, NetBeans, CodeBlocks
Contoh Text editor : Notepad, Visual Studio Code, Sublime text
