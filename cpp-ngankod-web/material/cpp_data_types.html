Tipe Data

dalam c++,, ada 6 jenis tipe data, yaitu:
int >> Memuat bilangan bulat(integer). Misal 1, 10, 44
float >> Memuat bilangan desimal. Misal 1.5, 4.22
double >> sama seperti float, namun memiliki range 2x lebih besar
string >> Memuat text. Misal "Hello, world". Untuk string selalu diapit oleh tanda "double quote"
char >> Memuat karakter. Misal 'a', 'A'. Untuk char selalu diapit oleh 'single quote'
bool >> Memuat Boolean yang hanya berisi 2 keterangan, true dan false
Untuk penggunaan tipe data akan dijelaskan di bagian Variabel
