Operator C++

Operator digunakan untuk melakukan operasi pada variabel.
dalam C++ ada 4 jeis operator, yaitu:
  1. Operator Aritmatika
  2. Operator Penugasan (Assignment)
  3. Operator Perbandingan (Comparison)
  4. Operator Logika

Operator Aritmatika

Operator Aritmatika adalah opereator dasar dalam matematika yang terdiri dari
Simbol Nama Fungsi
+ Penambahan Menambahkan value atau variabel
- Pengurangan Mengurangkan value atau variabel
* Perkalian Mengalikan Value atau variabel(yang berisi angka)
/ Pembagian Membagi nilai atau variabel(yang berisi angka)
% Modulus Menghitung sisa hasil bagi
++ Increment Kenaikan nilai variabel sebanyak 1
-- Decrement Penurunan nilai variabel sebanyak 1
Contoh :
int a = 10%3;
cout << a << endl; // 1
Dari kode diatas menghasilkan nilai 1 karena 10 dibagi 3 = 3 sisa 1. Dan operator % mengitung sisa hasil bagi.

Operator Penugasan

Operator Penugasan digunakan untuk menetapkan nilai ke variabel. Contoh:
int a = 10;
a += 10;
cout << a << endl; //a = a+10 = 20

Simbol Contoh Bentuk lain
= x = 5 x = 5
+= a+=5 a = a+5
-= a -= 5 a -= a%5
/= a /= 5 a = a/5
*= a *= 5 a = a*5
%= a %= 5 a = a%5

Operator Perbandingan

Operator Perbandingan digunakan untuk membandingkan 2 nilai. Contoh:
int a , b;
a = 10;
b = 10;
cout << a == b << endl; // 1(true) karena nilai a bernilai sama dengan b
Simbol Nama Contoh
== sama dengan x == 5
!= Tidak sama dengan a != b
< Kurang dari a < 5
> Lebih dari a > 5
<= Kurang dari sama dengan a <= 5
>= Lebih dari sama dengan a >= 5

Operator Logika

Operator Logika digunakan untuk melakukan perbandingan logika antara 2 variabel. Contoh:
int a, b;
a = 10;
b = 15
cout a > 5 && b < 20; //menghasilkan true karena nilai a adalah 10 (lebih dari 5) dan nilai b adalah 15(kurang dari 20)
Simbol Nama Penjelasan
&& AND LOGIC Menghasilkan nilai true jika kedua pernyataan bernilai benar
|| OR LOGIC Menghasilkan nilai true jika salah satu pernyataan bernilai benar
!= NOT LOGIC Membalikkan hasil operasi logika, bernilai benar jika hasilnya salah dan sebaliknya
