Variabel

Variabel digunakan untuk menyimpan data

Aturan penulisan Variabel

1. Variabel bisa terdiri dari gabungan angka, huruf, dan tanda underscore ( _ )
2. Dilarang diawali dengan angka.
3. Hanya terdiri dari satu kata, jika ingin memisahkan 2 kata sebaiknya gunakan underscore atau penulisan camelCase. Contoh nama_variabel atau namaVariabel.
4. Dilarang mengandung keyword yang digunakan di bahasa pemrograman seperti for, return, break, dsb.

Pendeklarasian Variabel

Syntax untuk mendeklarasikan variabel
tipe_data namaVariabel = value
Contoh :
int a = 15;
cout << a << endl;

Kalian juga bisa mendeeklarasikan nilai variabel tanpa perlu memasukkkan nilainya langsung. Contoh :
int a;
a = 15;
cout << a << endl;

Mendeklarasikan lebih dari 1 Variabel

Untuk beberapa tipe variabel dengan tipe data yang sama, sebaiknya dibuat dalam 1 baris. Gunakan ',' (comma) sebagai pemisah antar variabel. Contoh :
int a = 15, b = 20;
string namaDepan = "John", namaBelakang= "Doe";
Dapat juga berbentuk seperti ini :
int a, b, c;
a = 10;
b = 15;
c = 8;
