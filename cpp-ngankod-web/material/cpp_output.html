Output (menampilkan text)

Untuk menampilkan text pada C++ kita gunakan cout yang dibarengi dengan operator "<<"
//program pertamaku
#include <iostream>
using namespace std;
int main(){
  cout << “belajar bahasa c++”;
  return 0;
}

keyword cout bisa ditulis secara berulang. contoh:
cout << "belajar c++";
cout << "NganKod HIMTI";
Namun, jika penulisannya seperti contoh diatas akan menghasilkan output belajar c++Ngankod HIMTI.
Nah ada 2 cara membuat baris baru di C++.
1. Menggunakan endl;. Contoh:
cout << "belajar c++" << endl;
cout << "NganKod HIMTI" << endl;

2. Menggunakan "\n"; . Contoh:
cout << "belajar c++\n";
cout << "NganKod HIMTI\n";
