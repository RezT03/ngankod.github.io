C++ Syntax

Cara terbaik belajar pemrogrman adalah dengan memprektekkannya langsung. Jangan hanya sekedar menghafal syntax atau aturan penulisan, namun juga harus sering dipraktekkan karena beda bahasa pemrograman beda pula syntax-nya.
//program pertamaku
#include <iostream>
using namespace std;
int main(){
  cout << “belajar bahasa c++”;
  return 0;
}
Penjelasan
Baris 1 terdapat tanda “//” adalah komentar, komentar tidak akan dibaca oleh komputer, komentar berfungsi untuk mempermudah programmer tentang fungsi kode yang dibuat. Komentar yang diawali dengan // hanya untuk 1 baris, namun untuk lebih dari satu baris menggunakan tanda /*.... */ . Contoh :
/*blok
Komentar*/
Baris 2 terdapat #include <iostream> adalah pustaka(library) file header yang memungkinkan kita bekerja dengan keyword input dan output, seperti cout(pada baris 5). Masing-masing library memiliki fungsi yang berbeda.
Baris 3 terdapat using namespace std; berarti kita dapat menggunakan nama untuk keyword dan variabel dari library standar. Untuk header #include <iostream> dan using namespace std; selalu muncul hampir di semua program C++.
Baris 4 int main(){..} adalah sebuah fungsi yang selalu muncul di C++. Semua yang ada didalam tanda kurung kurawal { } akan dijalankan.
Baris 5 cout adalah perintah untuk menampilkan apa yang ada setelah tanda operator insertion ( <<).
Baris 6 return 0; berfungsi untuk mengakhiri fungsi utama. Note : Perlu diingat, untuk setiap pernyataan di C++ harus diakhiri dengan tanda titik-koma (semicolon) ; .
