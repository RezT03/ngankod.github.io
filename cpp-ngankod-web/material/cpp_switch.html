Switch Case

Switch case memiliki fungsi yang sama dengan if namun memiliki syntax yang berbeda.
switch(kondisi){
  case a:
 //blok kode
break;
  case b:
 //blok kode
break;
default: //dijalankan jika tidak ada opsi yang memenuhi
}
Contoh:
int bulan = 2;
switch (bulan) {
 case 1:
 cout << "Januari";
 break;
 case 2:
 cout << "Februari";
 break;
 case 3:
 cout << "Maret";
 break;
 case 4:
 cout << "April";
 break;
 case 5:
 cout << "Mei";
 break;
 default:
 cout << "tidak ditemukan";
}
Pada contoh kode diatas akan menghasilkan Februari karena opsi yang memenuhi adalah case 2.
