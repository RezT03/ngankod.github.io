Pengkondisian

Dalam sebuah proses program, biasanya terdapat kode penyeleksian kondisi, kode pengulangan program, atau kode untuk pengambilan keputusan. Hal ini pernah kalian pelajari pada mata kuliah Computational Thinking materi flowchart

Untuk menjalankan perintah tersebut, ada beberapa keyword yang bisa digunakan :
  1. if - Menjalankan blok kode jika kondisi benar
  2. else - Menjalankan blok kode jika kondisi salah
  3. else if - Menentukan kondisi baru jika kondisi pertama salah
  4. switch - Memiliki lebih banyak opsi yang bisa dijalankan, namun hanya menjalankan yang benar

if

if (conditions){
//blok program yang akan dijalankan
}
misal disini kita akan menentukan apakah 10 lebih kecil dari 20.
if (10 < 20){
cout << "10 lebih kecil dari 20";
}
Kode diatas akan menjalankan blok kode dibawah if karena kondisinya terepenuhi

else

if (conditions){
//blok program yang akan dijalankan
} else { //akan dijalankan ketika tidak terpenuhi }
misal disini kita akan menentukan apakah 10 lebih kecil dari 20 namun menggunakan operator yang salah.
if (10 > 20){
cout << "10 lebih kecil dari 20";
} else{
cout << "20 lebih besar dari 10";
}

else if

if (conditions){
//blok program yang akan dijalankan
} else if(kondisi_baru){
//akan dijalankan ketika kondisi baru terpenuhi
} else { }
