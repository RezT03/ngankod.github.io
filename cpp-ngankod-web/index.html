Selamat datang di program NganKod(Ngaji Koding) HIMTI UNSIQ

NganKod merupakan program kerja Departemen Pengembangan Sumber Daya Mahasisa (PSDM) HIMTI UNSIQ. NganKod bertujuan untuk membantu mahasiswa untuk belajar dasar-dasar pemrograman karena pembelajaran di dalam kelas waktunya sangat terbatas dan juga cukup sulit untuk berinteraksi dengan dosen di luar kelas.
